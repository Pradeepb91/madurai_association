# 🏛️ Madurai New LIG Welfare Association Website

A complete Association Management Web Application built using **Python Django** for the Madurai New LIG Welfare Association.

This project helps residents and administrators manage members, complaints, suggestions, reports, and association activities through a modern web interface.

# ✨ Features

## 🏠 Home Page
- Responsive Bootstrap UI
- Association introduction
- Image slider
- Latest updates
- Quick navigation

---

## 👥 Members Management

- Add/Edit/Delete members
- Member designation
- Member photo
- Display order
- Active/Inactive members
- Cloudinary image storage

---

## 🖼 Gallery

- Event based gallery
- Multiple images per event
- Photo captions
- Cloudinary image hosting

---

## 📄 Reports

- Upload Association Reports (PDF)
- Download/Open reports
- Report search
- Upload date

---

## 📢 Complaint Management

Residents can submit complaints with:

- House Number
- Name
- Mobile Number
- Complaint Category
- Complaint Description
- Photo Upload
- Auto Complaint Number

Example:

MNL-CP-2026-0001

Complaint Status:

- Pending
- In Progress
- Resolved
- Rejected

---

## 💡 Suggestion Management

Residents can submit suggestions.

Includes:

- Suggestion Number
- Subject
- Description
- Attachment
- Email
- Mobile Number

Example:

MNL-SG-2026-0001

---

## 🔐 Admin Panel

Custom Django Admin

Manage:

- Members
- Reports
- Gallery
- Complaints
- Suggestions

# ⚙ Installation

Clone Repository

```bash
git clone https://github.com/Pradeepb91/madurai_association.git
```

Move to project

```bash
cd madurai_association
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Install Packages

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py migrate
```

Run Server

```bash
python manage.py runserver
```
